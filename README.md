# How to see the App ? 
1. Clone the repository 
```sh
git clone https://github.com/yogita10/Home_Rental_App.git
```
3. Go to the clone folder
```sh
cd Home_Rental_App
```
5. Run the following commands: 
```sh
 npm install 
```
```sh
 npm run android 
```
